# Interactively Set Github PAT

This wraps gitcreds::gitcreds_set() and adds a system call to edit the
user global git config to set the cache timeout to 1 billion seconds if
running a Linux system.

## Usage

``` r
gitcreds_set()
```

## Value

nothing
